# hybrid_charge_sensing

Python project for numerical simulations, experimental data analysis, plotting, and Jupyter notebooks related to hybrid charge sensing.

## Setup on macOS

Create the virtual environment:

```bash
python3.12 -m venv .venv
```

Activate it:

```bash
source .venv/bin/activate
```

Upgrade pip:

```bash
python -m pip install --upgrade pip
```

Install the project in editable mode:

```bash
python -m pip install -e .
```

Run the charge calculations module:

```bash
python src/hybrid_charge_sensing/charge_calculations.py
```

Start JupyterLab:

```bash
jupyter lab
```
